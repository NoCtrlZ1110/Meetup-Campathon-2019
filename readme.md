# Basic usage
## Installation
### For Frontend
Install with [Expo-client](https://docs.expo.io/versions/latest/)
```bash
npm install -g expo-cli
```
```bash
cd Frontend
```
```bash
npm install
```
```bash
expo start
```

### For Backend
Install with [Nodemon](https://nodemon.io/)
```bash
npm install -g nodemon
```
```bash
cd Backend
```
```bash
cd npm install
```
``` bash
npm start
```
